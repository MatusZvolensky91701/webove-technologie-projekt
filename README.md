# MeCargo - Order Tracking Portal

This project is a static web portal for tracking orders.

## Pages
- Home page
- Track Order
- Statuses
- Gallery
- Contact
- Orders

## Technologies used
- HTML5
- CSS3
- JavaScript
- JSON
- AJAX using fetch()

## How to run the project
1. Download the project folder.
2. Open `index.html` in a browser.
3. For AJAX functionality, run the project using a local server, for example Live Server in Visual Studio Code.
